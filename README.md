# vntt  
改良vntt使用的一些油猴脚本  
https://greasyfork.org/zh-CN/scripts/448572-vntt%E7%BF%BB%E8%AF%91%E8%BE%85%E5%8A%A9  
  
## 主要功能  
**· 项目查重（new）**  
主动提醒项目中该原文是否已经有现成翻译，避免多人合作时翻译不统一

**· 机器翻译**  
已接入 ChatGPT翻译、Mirai翻译、谷歌翻译、百度翻译、腾讯翻译  
  
**· 翻译记忆**  
可以记忆使用者的所有翻译句子，并在翻译新句子时提供类似于Trados和MemoQ的智能填充  
  
**· 句子成分分析**  
匹配术语表和人名表，提供点击插入按钮  
  
**· 复制原文**
